# smart-framework

Smart framework is a framework which is able to deploy, in a smart way, the appropriate code to a heterogeneous wireless sensor network (e.g. smartphones, sensor nodes etc.) and to retrieve sense data from the nodes of the network.

The repository comprehends an Android application to test the framework; this latter which is able to talk with the available sensor nodes in the network and deploy the appropriate software on the nodes (Android app on the smartphone, TinyOS aplication on TinyOS node and ContikiOS application on ContikiOS node).
